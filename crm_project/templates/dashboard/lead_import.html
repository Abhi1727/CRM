Import Instructions

📁 Supported File Formats

  • CSV files (.csv)
  • Excel files (.xlsx, .xls)
  • Maximum file size: 10MB

📋 Required Columns

  • name - Lead's full name
  • mobile - Primary mobile number
  • email - Email address

🔧 Optional Columns

  • address - Street address
  • city - City
  • state - State/Province
  • postalcode - Postal code
  • country - Country
  • status - Lead status (default: 'lead')
  • course_name - Course/product name

⚠️ Important Notes

  • Duplicate mobile numbers or emails will be skipped
  • Invalid dates will be ignored
  • New leads will be unassigned by default - you can assign them later
  • Make sure your file has proper headers in first row
  • Encoding Issues: If you get encoding errors, try saving your CSV as UTF-8
  • File Format: CSV files should use comma (,) as separator
  • Special Characters: Avoid special characters in mobile numbers
