📋 How to Use Bulk Assignment

  1. Go to the Leads List
  2. Select leads using the checkboxes in the first column
  3. Click the "Bulk Assign" button that appears
  4. Choose the user to assign leads to
  5. Add optional assignment notes
  6. Click "Assign Leads" to complete the process
📌 Note: You can only assign leads to users within your hierarchy. The system will automatically validate permissions for each lead.
